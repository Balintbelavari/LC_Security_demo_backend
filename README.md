This demo website is set up to showcase a text message scam detection model.

To run the webapp, start Frontend with moving to frontend folder and starting React app:
cd frontend
npm start

If you are happy with how the frontend looks, build the changes:
npm run build

Than deploy the changes made with restarting Heroku hosting:
git push heroku main

Wait until heroku deploys changes, than view website at: 
https://lc-security-backend-d51e9de3f86b.herokuapp.com
